# Code of Conduct

## Our Standard

This project is intended to be a respectful, constructive environment for collaboration.

Examples of expected behavior:

- be respectful in discussions and reviews
- assume good intent and focus on technical substance
- give actionable feedback
- welcome contributors with different backgrounds and experience levels

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory or exclusionary language
- bad-faith disruption of project discussions
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this code of conduct.

## Reporting

Report conduct issues to the project maintainers through a private channel appropriate for the repository hosting platform.
